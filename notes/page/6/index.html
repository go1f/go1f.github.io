XSS和CSRF的不同点。 XSS是跨站脚本攻击，利用的是用户对链接/网页的信任，加载恶意脚本窃取用户信息、钓鱼等。 CSRF是跨站请求伪造，XSS可以是它的一种手段，利用的是服务端对浏览器的信任，借助用户的身份认证像Cookie，造成信息篡改、发帖、蠕虫等。 你知道同源策略吗？都有哪些。 域名相同、端口相同、协议相同，表单提交、重定向、链接跳转不受限；外域资源引用可以的，像img、script、css标签 CSP Content-Secure-Policy 允许内容来自的域 排序算法？ 冒泡、快速、选择、插入、桶排序 计算机网络协议模型？ 物理层、数据链路层、网络层、传输层、会话层、表示层、应用层 TCP、UDP不同点？ TCP面向连接，可以做流量控制、提供稳定可靠连接 UDP无连接、面向报文，优点快速，适合视频传输、DNS 你平时写脚本都用到什么库，什么工具？ requests、os、json、PyV8；sublime、VS Code XSS的防御方法。 过滤用户输出，如果输出点在标签属性位置，注意设置函数，htmlspecialchars()默认不转义单引号 富文本编辑器如何过滤XSS payload 白名单，正则匹配，非白一律转义。 如何判断一个URL是否有SQL注入 单双引号，测试使用未过滤字符 '&&'0 、1'&&sleep(1)&&'1 mysql UDF 提权 phpstudy使用32位的MySQL，利用 SQLmap的UDF，放在mysql/lib/plugin文件夹 cloak.py -d -i C:\tools\sqlmap\data\udf\mysql\windows\32\lib_mysqludf_sys.dll_ select @@basedir select unhex() into outfile 'xxx/mysql/lib/plugin/myudf.so' create function sys_eval returns string soname 'lib_mysqludf_sys. …
