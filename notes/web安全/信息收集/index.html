Shodan的http.favicon.hash语法详解与使用技巧

环境要求 python2.7 VCForPython27.msi pip mmh3 requests

#iconhash.py
import mmh3
import requests
 
response = requests.get('http://127.0.0.1/favicon.ico')
favicon = response.content.encode('base64')
hash = mmh3.hash(favicon)
print(hash)

py -2 iconhash.py
