网站个人信息

定制社工字典

工具： cupp、pydictor

相关信息

  • 域名(子域名) =》 Whois(联系方式、手机邮箱) =》 IP(地址) =》IP反查多个域名=》循环

  • 员工信息、组织框架、企业法人、企业综合信息、工商注册、新闻组/论坛、招聘网站

  • 员工信息：姓名、工号、家庭、交际信息、上网习惯（豆瓣、知乎、微博等社交平台）

  • 社交账号信息：qq、qq群、微博、微信、支付宝、员工邮箱账号

  • Metadata: 图片Exif

网络搜索引擎

google

语法

intitle:/allintittle: 标题

intext:/alltext: 正文

inanchor:/allinanchor: 链接锚点

site: 域名

inurl:/allinurl: URL

link: 跳转至该网站的网页

cache: 缓存快照

filetype: 文件类型 例如 log日志 mdb数据库 txt

-关键字 忽略该关键字

"字符串" 强制匹配

. 匹配一个字符

* 匹配任意字符

条件间空格隔开，默认表示 &与运算 ；使用 |表示 或运算

Shodan的http.favicon.hash语法详解与使用技巧

环境要求 python2.7 VCForPython27. msi pip mmh3 requests

#iconhash.py
import mmh3
import requests
 
response = requests.get('http://127.0.0.1/favicon.ico')
favicon = response.content.encode('base64')
hash = mmh3.hash(favicon)
print(hash)

py -2 iconhash.py

Recon-NG框架

#下载所有模块
marketplace install all

#启动
recon-ng -w <project name>    #新建工程启动
recon-ng -r <filename>    #读取工程文件启动
#配置代理http proxy
options set PROXY 192.168.145.1:10808

#搜索模块
modules search google
#google 获取域名
modules load recon/domains-hosts/google_site_web
options set SOURCE bilibili.com
run
show hosts
#然后域名解析
modules load recon/hosts-hosts/resolve
options set SOURCE query select host from hosts
#输出报告
modules load reporting/html
